# todo-hono-postmark

**Generated**: 2025-04-09
**Scope**: Turborepo monorepo with Cloudflare deployment

## Overview

Full-stack TypeScript application with React frontend, Hono backend, Drizzle ORM, Better-Auth, deployed via Alchemy to Cloudflare Workers.

## Tech Stack

| Layer | Technology |
|-------|------------|
| Runtime | Cloudflare Workers |
| Package Manager | Bun |
| Frontend | TanStack Router, Tailwind v4, shadcn/ui |
| Backend | Hono, Zod OpenAPI |
| Database | PostgreSQL via Neon, Drizzle ORM |
| Auth | Better-Auth |
| Email | Postmark |
| Lint/Format | Oxlint, Oxfmt |
| Deploy | Alchemy (Cloudflare) |

## Conventions

- **Oxlint/Oxfmt**: Used instead of ESLint/Prettier. Run `bun run check` before committing. **IMPORTANT**: Always use `bunx oxlint` instead of `oxlint` directly, as oxlint is not installed globally.
- **Tailwind v4**: Configured in Vite plugin (no tailwind.config.js). CSS vars in `apps/web/src/index.css`.
- **Workspace imports**: Use `@todo-hono-postmark/*` for cross-package deps.
- **Catalog deps**: Root `package.json` defines shared versions via `"catalog:"`.
- **File-based routing**: TanStack Router auto-generates routes from `apps/web/src/routes/`.
- **Drizzle ORM**: Snake_case casing, migrations in `packages/db/src/migrations/`.

## Anti-Patterns

- **NEVER edit** `apps/web/src/routeTree.gen.ts` - auto-generated by TanStack Router.
- **No `.api` package** - API logic is in `apps/server/`, not a shared package.
- **No Prettier** - Use Oxfmt for formatting.
- **No ESLint** - Use Oxlint for linting (via `bunx oxlint`).
- **No vitest.config.ts** - Using default vitest configs (not yet configured).

## Commands

```bash
# Install deps
bun install

# Dev (runs infra + db:studio)
bun run dev

# Dev specific apps
bun run dev:web       # Frontend only
bun run dev:server    # Backend only

# Database
bun run db:push       # Push schema changes
bun run db:studio     # Drizzle Studio UI
bun run db:generate   # Generate migrations
bun run db:migrate    # Run migrations

# Build & Deploy
bun run build         # Build all packages
bun run deploy        # Deploy via Alchemy
bun run destroy       # Destroy Alchemy resources

# Check
bun run check-types  # TypeScript check
bun run check        # Oxlint + Oxfmt
bunx oxlint          # Run oxlint directly (required - oxlint not in PATH)
bunx oxfmt --write   # Run oxfmt directly (required - oxfmt not in PATH)
```

## Gotchas

- **Env files required**: `.env` and `.env.local` needed in multiple locations (apps/server/, apps/web/, packages/infra/). See `.env.example` files.
- **Cloudflare Workers**: Server runs on Workers runtime - use Workers-compatible APIs.
- **Alchemy**: Infrastructure as code for Cloudflare. Stateful resources defined in `packages/infra/src/`.
- **Generated routes**: TanStack Router generates `routeTree.gen.ts` - don't import from it directly.
